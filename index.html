Instrucciones

1. Selecciona el tipo de factura que deseas analizar (ingreso, egreso, pago, etc.).

2. Elige si la factura es emitida o recibida.

3. Marca los datos que deseas extraer (Folio Fiscal, RFC, IVA, etc.).

4. Sube tus archivos XML arrastrándolos o seleccionándolos con el botón.

5. Presiona "Analizar XML y Descargar Excel" para generar el archivo.
